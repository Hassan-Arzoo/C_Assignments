# 📚 C Programming Assignments

Welcome to my **C Programming Assignments** repository!

This repository contains my C programming assignments completed as part of my learning journey. Each assignment consists of multiple programming questions designed to strengthen core programming concepts and problem-solving skills.

## 📂 Repository Structure

```text
C-Programming-Assignments/
│── Assignment-1/
│── Assignment-2/
│── Assignment-3/
│── Assignment-4/
│── ...
└── README.md
```

Each assignment folder contains the source code (`.c` files) for the questions included in that assignment.

## 🎯 Topics Covered

* Basic Input & Output
* Variables and Data Types
* Operators
* ASCII Characters
* Conditional Statements
* Loops
* Pattern Printing
* Arrays
* Strings
* Functions
* Pointers
* Structures
* File Handling
* Miscellaneous Programming Problems

## 🛠️ Requirements

* GCC Compiler
* Any C IDE or Code Editor (VS Code, Code::Blocks, Dev-C++, etc.)

Compile and run a program:

```bash
gcc filename.c -o program
./program
```

## 🎓 Purpose

* Practice C programming fundamentals.
* Improve logical thinking and problem-solving skills.
* Maintain a structured record of assignment solutions.
* Create a reference repository for future learning and revision.
